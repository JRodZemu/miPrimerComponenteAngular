# 🧩 Mi Primer Proyecto Angular

Este es mi primer proyecto desarrollado con [Angular](https://angular.io/), como parte de mi aprendizaje en este framework. El objetivo es crear componentes reutilizables y entender la estructura de una aplicación Angular moderna.

## 🛠️ Tecnologías utilizadas

- **Angular** - Framework frontend
- **TypeScript** - Lenguaje principal
- **HTML/SCSS** - Maquetación y estilos

## 📋 Requisitos previos

Para ejecutar este proyecto en tu ordenador, necesitarás tener instalado:

- **Node.js** (versión 18 o superior)
- **Angular CLI** - Puedes instalarlo globalmente con `npm install -g @angular/cli`

## 🚀 Cómo empezar

Sigue estos pasos para tener el entorno de desarrollo funcionando:

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/JRodZemu/miPrimerComponenteAngular.git
   cd miPrimerComponenteAngular
   ```
2. **Instala las dependencias**
 ```bash
   npm install
   ```
3. **Ejecuta el servidor**
   ```bash
   ng serve
   ng serve -o
   ```
